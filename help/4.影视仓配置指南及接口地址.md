## 配置步骤

### 步骤一：下载安装影视仓

1. 从TV应用商店下载最新版本的影视仓
2. 安装到您的电视盒子上
3. 打开影视仓应用

### 步骤二：添加配置源

#### 影视仓TV版

1. 打开APP
2. 进入设置
3. 点击配置地址
4. 手动输入接口地址
5. 点击确定，等待加载即可

#### 影视仓手机版

1. 打开APP
2. 点击右上角"+"号
3. 点击弹窗右上角"添加"
4. 填写名称、粘贴接口地址
5. 点击确定，然后等待加载即可

### 步骤三：使用影视仓

1. 返回主界面，选择您喜欢的分类
2. 浏览并选择您想观看的内容
3. 点击播放，选择合适的播放源
4. 开始观看

## 常见问题

### 配置源无法加载？

1. 检查网络连接是否正常
2. 确认配置地址是否正确
3. 尝试更换其他配置源
4. 检查影视仓版本是否最新

### 播放时没有声音？

1. 检查设备音量是否开启
2. 尝试切换播放器内核
3. 检查播放源是否有声音
4. 重新安装影视仓
