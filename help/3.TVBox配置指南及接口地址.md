## TVBox配置步骤

### 步骤一：下载安装TVBox

1. 从TV应用商店下载最新版本的TVBox
2. 将APK文件复制到U盘或通过网络传输到电视盒子
3. 在电视盒子上安装TVBox
4. 安装完成后，打开TVBox应用

### 步骤二：添加配置源

#### 方法1：手动输入配置地址

1. 打开TVBox应用
2. 进入设置页面（通常在左上角或右上角的菜单中）
3. 找到"配置地址"或"添加配置源"选项
4. 手动输入配置接口地址
5. 点击"确定"或"保存"，等待加载完成
6. 返回主界面，即可看到配置源提供的内容

#### 方法2：扫描二维码添加

1. 打开TVBox应用
2. 进入设置页面
3. 找到"扫描二维码"选项
4. 将二维码对准电视盒子的摄像头
5. 扫描成功后，自动加载配置源
6. 返回主界面，即可看到配置源提供的内容

#### 方法3：本地文件配置

1. 将配置文件（通常为.json格式）复制到U盘或电视盒子的存储中
2. 打开TVBox应用
3. 进入设置页面
4. 找到"本地配置"或"从文件导入配置"选项
5. 选择存储中的配置文件
6. 等待加载完成，返回主界面即可使用

## TVBox高级配置

### 1. 切换播放器内核

TVBox支持多种播放器内核，可以根据需要切换：

1. 进入TVBox设置页面
2. 找到"播放器"或"内核设置"选项
3. 选择合适的播放器内核（如ExoPlayer、IJKPlayer等）
4. 保存设置，重启应用生效

### 2. 自定义皮肤

TVBox支持自定义皮肤，可以根据个人喜好更换：

1. 下载喜欢的皮肤文件（通常为.zip格式）
2. 将皮肤文件复制到TV盒子的存储中
3. 进入TVBox设置页面
4. 找到"皮肤设置"选项
5. 选择本地皮肤文件，应用即可

### 3. 添加自定义源

如果您有自己的视频源，可以添加到TVBox中：

1. 进入TVBox设置页面
2. 找到"自定义源"或"添加源"选项
3. 输入源名称和源地址
4. 保存设置，返回主界面即可看到自定义源

## TVBox常见问题解决

### 问题1：配置源无法加载

**解决方法**：
- 检查网络连接是否正常
- 确认配置地址是否正确
- 尝试更换其他配置源
- 检查配置源是否已失效，可在本文获取最新的配置源

### 问题2：播放时卡顿或缓冲时间长

**解决方法**：
- 检查网络连接速度
- 切换到其他播放源
- 降低视频画质设置
- 清除应用缓存和数据
- 重启路由器和设备

### 问题3：播放时没有声音

**解决方法**：
- 检查设备音量是否开启
- 尝试切换播放器内核
- 检查播放源是否有声音
- 重新安装TVBox

### 问题4：应用闪退或无法打开

**解决方法**：
- 确认TVBox与电视盒子的Android版本兼容
- 尝试清除应用数据和缓存
- 卸载并重新安装TVBox
- 检查设备是否有足够的内存
